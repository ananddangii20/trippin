# Trippin ✈️

Trippin is a mobile application built with React Native and Firebase that helps users organize and manage trips with friends.

## Features

* User Authentication
* Create and Manage Trips
* Realtime Group Chat
* Trip Budget Tracking
* Group Member Management
* Shared Trip Photos
* Realtime Data Synchronization

## Tech Stack

* React Native (Expo)
* Firebase Authentication
* Cloud Firestore
* Firebase Storage
* React Navigation

## Configuration

Set up a Firebase project and update the Firebase configuration in:

`src/services/firebase.js`
